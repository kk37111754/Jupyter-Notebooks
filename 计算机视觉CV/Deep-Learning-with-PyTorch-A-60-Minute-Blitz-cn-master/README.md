## 1、简介

此教程的目标：

* 更高层次地理解Pythrch的Tensor库以及神经网络。
* 训练一个小的神经网络模型用于分类图像。

本教程假设读者对`numpy`有基本的了解


## 2、环境

* PyTorch版本0.4及以上（[PyTorch 1.0 **稳定版**](https://pytorch.org/get-started/locally/)已经发布，还有什么理由不更新呢~）
* [torchvision](https://github.com/pytorch/vision) 0.2.1
